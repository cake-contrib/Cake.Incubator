GetMatchingFiles(ICakeContext, IEnumerable<FilePath>) Method

Summary

Returns files in the same directory that have the same file name but different extensions
Namespace
Cake.Incubator
Containing Type
FileExtensions

Syntax

[CakeMethodAlias]
public static IEnumerable<IFile> GetMatchingFiles(this ICakeContext context, IEnumerable<FilePath> files)

Examples

Locates files with the same name in the same directory, but different extensions. The .pdb to your .dll as it were. 
 // /output/file.dll
 // /output/file.xml
 // /output/file.pdb
 // /output/another.dll

 IEnumerable<FilePath> matchingFiles = GetMatchingFiles(new FilePath("/output/file.dll"));

 matchingFiles[0]; // /output/file.xml
 matchingFiles[1]; // /output/file.pdb
